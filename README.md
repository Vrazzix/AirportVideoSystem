# ✈️ Aircraft & Human Detection System

Веб-приложение на Streamlit для обработки видео тремя моделями YOLOv8:

1. **🛞 Колёса и башмаки** — детекция колёс и башмаков самолёта
2. **🧑 Люди** — детекция людей в кадре
3. **🦴 Поза** — определение позы человека по ключевым точкам (YOLOv8-pose)

## Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Структура моделей

Создайте папку `models/` и поместите туда ваши `.pt` файлы:

```
app/
├── app.py
├── requirements.txt
├── models/
│   ├── wheels_chocks.pt   ← модель колёс и башмаков
│   ├── person.pt          ← модель детекции людей
│   └── pose.pt            ← модель YOLOv8-pose
```

> Пути к моделям можно изменить в боковой панели приложения.

### 3. Запуск

```bash
cd app
streamlit run app.py
```

Приложение откроется в браузере по адресу `http://localhost:8501`.

### 4. Для воспроизведения видео в браузере

Убедитесь, что установлен `ffmpeg`:

```bash
# Ubuntu/Debian
sudo apt install ffmpeg

# macOS
brew install ffmpeg

# Windows (через choco)
choco install ffmpeg
```

## Возможности

- Загрузка видео (MP4, AVI, MOV, MKV, WMV)
- Включение/выключение каждой модели отдельно
- Настройка порога уверенности и IoU
- Пропуск кадров для ускорения (каждый N-й кадр)
- Визуализация в реальном времени во время обработки
- Скачивание обработанного видео
- Кэширование моделей между запусками
